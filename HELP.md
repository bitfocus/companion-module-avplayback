## IfElseWare AV-Playback

**Instance Settings

* UDP Address
* Port Number
* Use Pad Fix - Appending a space to end of command string breaks compatability with some builds of AV-Playback. 

**Available commands for IfElseWare AV-Playback**

* Play
* Pause
* Stop
* Play preview clip
* Preview Pause
* Preview Stop
* Next Clip
* Previous Clip
* Freeze temp
* Loop temp
* Next Tag
* Previous Tag
* Goto 10
* Goto 20
* Goto 30
* Goto 60
* Load Clip (number)
* Play Clip (number)
* Goto (Timecode)
